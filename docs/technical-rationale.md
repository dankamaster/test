# Technical Rationale

## Chosen stack
Plain HTML, CSS, and JavaScript.

## Why
- The assignment requires a standalone HTML page, not CMS.
- Static files are fastest and easiest to deploy to GitHub Pages, Netlify, or Vercel.
- No framework reduces Lighthouse risk.
- Clean semantic HTML supports SEO, accessibility, and AI extraction.
- PWA basics are enough to demonstrate production awareness without overengineering.
