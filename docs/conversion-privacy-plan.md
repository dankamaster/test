# Conversion and Privacy Plan

## Primary conversion

Email-first documentation request.

## Popup

Headline: Request a COI Documentation Scope by Email

Fields:

- Work email
- Country or region
- Document needed
- Deadline
- Message

Privacy microcopy:

We use your email only to respond to this documentation request. Do not include sensitive personal identifiers, confidential client details, privileged information, or case-specific facts unless your legal team has approved this communication method.

## Marketing consent

The legal documentation request form does not include a marketing opt-in. It is used only for documentation-scope contact.

If newsletter or marketing consent is added later, it should be handled in a separate flow and not bundled with the legal documentation request form.

## Cookie plan

No analytics or advertising cookies are loaded by default. If analytics are added later, load only after consent and provide accept/reject/manage options.
